Jika form observasi akan dimasukkan ke **website/aplikasi digital** dan hasilnya langsung berupa **angka**, sebaiknya dibuat lebih sederhana dengan **skor per indikator** tanpa kolom BB-MB-BSH-BSB.

Tema:

- Eksperimen sawi berubah warna
- Eksperimen jagung menari
- Eskperimen uji asam basa
- Eksperimen fotosintesis
- Eksperimen pengamatan serangga dan serbuk sari bunga

## Form Observasi Digital Anak

**Kegiatan:** VFT Experiment Natural Science dan Eksperimen STEAM EduGreen

**Nama Anak:** …………………
**Tema Eksperimen:** …………………

**Petunjuk Pengisian:**

Pilih skor perkembangan anak:

- **1 = Belum Berkembang (BB)**
- **2 = Mulai Berkembang (MB)**
- **3 = Berkembang Sesuai Harapan (BSH)**
- **4 = Berkembang Sangat Baik (BSB)**

| No | Indikator Observasi | Skor (1–4) |
| --- | --- | --- |
| 1 | Anak memperhatikan video Experiment Natural Science dengan fokus | |
| 2 | Anak menunjukkan rasa ingin tahu melalui pertanyaan atau tanggapan | |
| 3 | Anak mampu mengamati perubahan yang terjadi dalam eksperimen | |
| 4 | Anak mampu membuat prediksi sederhana sebelum eksperimen | |
| 5 | Anak mampu mengikuti langkah-langkah eksperimen sesuai arahan | |
| 6 | Anak mampu menggunakan alat dan bahan eksperimen sederhana dengan aman | |
| 7 | Anak mampu menjelaskan hasil eksperimen secara sederhana | |
| 8 | Anak mampu menghitung/mengelompokkan/membandingkan bahan dalam eksperimen | |
| 9 | Anak mampu menceritakan kembali proses kegiatan yang dilakukan | |
| 10 | Anak menunjukkan kreativitas dalam menyelesaikan kegiatan eksperimen | |
| 11 | Anak mampu menghubungkan eksperimen dengan fenomena alam sekitar | |
| 12 | Anak menunjukkan sikap aktif, percaya diri, dan bekerja sama selama kegiatan | |

## Output Sistem Website

Sistem dapat otomatis menghitung:

**Total Skor:** …… / 12
**Rata-rata:** ……

**Kategori:**

- 3,26–4,00 = Berkembang Sangat Baik (BSB)
- 2,51–3,25 = Berkembang Sesuai Harapan (BSH)
- 1,76–2,50 = Mulai Berkembang (MB)
- 1,00–1,75 = Belum Berkembang (BB)

Format ini lebih mudah karena guru hanya memilih angka **1–4**, kemudian website dapat otomatis menghasilkan:

- nilai rata-rata anak,
- grafik perkembangan,
- persentase capaian STEAM EduGreen,
- rekap kelas.
